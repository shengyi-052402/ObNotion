# JVM

当前 JVM 资料集中在运行时内存结构、类加载和垃圾回收这三条主线，适合作为 Java 运行时机制的入口。

## 正文

- 内存模型部分关注运行时数据区以及对象在内存中的组织方式。
- 类加载部分关注加载、链接、初始化以及相关触发条件。
- 垃圾回收部分关注回收算法、可达性分析和常见收集器思路。
- 这一主题与 Java 基础中的反射和类加载、并发中的内存可见性问题密切相关。
