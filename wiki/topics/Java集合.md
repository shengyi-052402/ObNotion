# Java集合

当前集合资料同时覆盖了集合框架结构和实际开发中的使用注意事项，适合一边掌握原理，一边形成编码习惯。

## 正文

- 结构层面，重点是 List、Set、Queue、Map 四类集合的特点与差异。
- 实践层面，重点是判空、遍历删除、去重、数组转换、`Collectors.toMap()` 等高频坑点。
- 集合主题虽然属于 Java 基础，但与并发问题、缓存数据结构、数据库结果映射都有直接联系。
