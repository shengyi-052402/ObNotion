# Java基础

当前仓库中的 Java 基础资料已经覆盖语言特性、面向对象、异常、泛型、反射、注解、SPI、序列化和 I/O，足以构成一条完整的基础复习主线。

## 正文

- 基础概念层面，重点是 Java 语言特性、JDK/JRE/JVM 关系、字节码与“编译+解释并存”的运行方式。
- 语法与对象模型层面，重点是基本数据类型、方法、面向对象三大特征、`Object`、`equals/hashCode`、`String`。
- 运行期能力层面，重点是异常、泛型、反射、代理、注解、SPI、序列化与 I/O。
- 这些内容会直接支撑集合、并发、JVM 和 Spring 的后续理解。
