# 操作系统

当前操作系统资料覆盖了进程线程、锁、内存管理和 I/O 等后端高频基础，是理解并发、网络和性能问题的重要底座。

## 正文

- 进程管理部分包括进程/线程差异、调度、上下文切换、通信方式。
- 内存管理部分包括虚拟内存、页表、段表、页面置换、写时复制等。
- I/O 部分包括多路复用、`select/poll/epoll` 和高性能网络服务基础。
