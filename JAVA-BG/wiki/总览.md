# 总览

这个仓库被组织成一个可持续维护的 Markdown Wiki，用来逐步沉淀和更新知识。

## 分层

- `raw/`：存放原始资料和待处理内容。
- `wiki/`：存放持续维护的知识页和综合结论。
- `system/`：存放流程说明和模板。

## 当前状态

- 仓库骨架已建立。
- 已经摄入一批后端基础与框架相关剪藏资料。
- 当前已形成 Java基础、Java并发、Java集合、JVM、MySQL、Redis、Spring生态、MyBatis、操作系统、计算机网络 10 个主题入口。
- 后续应继续把新资料放入 `Clippings/`，再逐步整理进 `raw/sources/` 和 `wiki/`。

## 下一步

- 将新增网页剪藏、论文、访谈记录或读书笔记放入 `Clippings/`。
- 按主题继续细化概念页，例如异常、反射、锁、事务、索引、TCP。
- 对资料较多的主题补充“学习路径”或“复习顺序”页面。
- 每次有实质更新时，同步维护 `wiki/索引.md` 和 `wiki/日志.md`。
